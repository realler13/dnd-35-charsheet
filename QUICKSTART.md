# Quick Start Guide

Get up and running with the D&D 3.5 Character Sheet in 60 seconds!

## 1. Start the Server

Open Terminal and run:

```bash
cd "/Users/denispetrochenkov/Desktop/1. Projects/D&D 3.5 CharSheet"
python3 -m http.server 8000
```

## 2. Open in Browser

Navigate to: **http://localhost:8000**

## 3. Create Your First Character

### Step-by-Step:

1. **Abilities Tab** (⚡ Start here!)
   - Select Race: Choose "Human" or "Elf"
   - Set Ability Scores: Click to edit STR, DEX, CON, INT, WIS, CHA
   - See racial bonuses apply automatically!

2. **Classes Tab**
   - Click "Add Level"
   - Choose Class: "Fighter" or "Wizard"
   - Enter HP (e.g., 10 for Fighter d10, 4 for Wizard d4)
   - Click "Save"
   - Skill points calculated automatically!

3. **Skills Tab**
   - Click "Edit Ranks" on any skill
   - Allocate your skill points
   - Class skills cost 1 point per rank
   - Cross-class skills cost 2 points per rank

4. **Character Tab**
   - View your complete character sheet!
   - All stats calculated automatically

5. **Save**
   - Click "Save" button in header
   - Character auto-saves every 5 seconds

## That's it! 🎉

Your character is created and saved!

## What's Next?

- **Add more levels:** Classes Tab → "Add Level"
- **Track wealth:** Inventory Tab → Wealth section
- **Add equipment:** Inventory Tab → Carried Items
- **Manage spells:** Spells Tab (for casters)
- **Log adventures:** Game Log Tab
